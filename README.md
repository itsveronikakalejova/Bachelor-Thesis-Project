# Sesh - Platform for Sharing Projects
---
## Login page
At the beginning, you will be greeted by a login page. If you don't have an account, you can easily create one by clicking the link below the "Login" button. Don't worry about security — passwords are stored in a hashed form.
![login.png](/figures/login.png)

---
## Projects page
When you sign in, you'll be taken directly to the Projects section. If you don't have any projects yet, you can create one, and it will be stored in the database. If you've already created projects, they will be displayed here.
![projects.png](/figures/projects.png)

---
## Singular project page
When you click on a project, the files linked to it will appear. Here, you can also create new files and add content to them. A key feature is that you can type on two devices simultaneously, and the changes will appear in real time on the other device.
![socketio.png](/figures/socketio.png)
